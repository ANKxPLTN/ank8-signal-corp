ANK8 System Architecture

Technical Overview and Infrastructure Design

Core Components

Learning Engine

The Learning Engine manages curriculum delivery, tracks user progress, and generates personalized recommendations. It stores and delivers learning modules while tracking completion and generating custom learning paths based on user goals and performance.

AI Coordination System

The AI Coordination System processes user queries and provides intelligent explanations. It operates on a read-only analysis model only, with no autonomous command execution. All suggestions are logged and auditable.

Server Agent

Deployed directly on user's VPS via: curl -sSL https://install.ankverdict.io/agent | bash

The agent can read system files, observe commands, and analyze configurations. It cannot modify files, execute commands autonomously, or install packages.

Data Storage Layer

Stores user data including account information, server configurations, learning progress, and custom goals. Infrastructure data includes hardware specifications, OS versions, and network status. All data is encrypted at rest using AES-256 encryption with key management via AWS KMS.

Audit and Compliance Logging

Every action is logged for accountability including authentication, API calls, and commands. Retention policies: 90 days hot storage, 1 year archive, 7 years legal hold.

Communication Protocol

User to Platform Communication

  • Protocol: REST API with WebSocket fallback
  • Authentication: API Key plus User JWT Token
  • Encryption: TLS 1.3 with perfect forward secrecy

Agent to Platform Communication

  • Health check every 5 minutes
  • Metrics push every 10 minutes
  • Event reporting in real-time
  • GZIP compression and batch operations

Security Architecture

Defense-in-Depth Strategy

  • Layer 1: Network Security - VPC isolation, DDoS protection
  • Layer 2: Application Security - Input validation, SQL injection prevention
  • Layer 3: Data Security - AES-256 at rest, TLS 1.3 in transit
  • Layer 4: Agent Security - Code signing verification, sandboxed execution

Technology Stack

  • Backend: Python 3.11+, FastAPI, PostgreSQL and Redis
  • Frontend: React 18+, Redux, Material-UI
  • Agent: Go compiled binary, less than 10MB
  • DevOps: Docker, Kubernetes, Terraform, GitHub Actions

Scalability Design

Horizontal scaling with stateless Learning Engine, load-balanced AI System, Kubernetes API Gateway, and sharded database. Target capacity: 10,000 concurrent users, 50,000 servers managed, 1 million queries per day, 99.9% uptime.

Disaster Recovery

RTO: 15 minutes. RPO: 5 minutes. Multi-region deployment with active-active API and automated failover. Triple redundancy for critical systems.
