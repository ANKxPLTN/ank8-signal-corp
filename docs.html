ANK8 System Architecture

Technical Overview & Infrastructure Design

Core Components

1. Learning Engine

Manages curriculum delivery, tracks user progress, generates recommendations.

2. AI Coordination System

Processes queries, provides explanations. Read-only analysis only, no autonomous execution.

3. Server Agent

Deployed on user's VPS via: curl -sSL https://install.ankverdict.io/agent | bash

Can: Read files, observe commands, analyze configurations

Cannot: Modify files, execute commands, install packages

4. Data Storage

User data encrypted at rest with AES-256. Infrastructure data includes hardware specs.

5. Audit Logging

Every action logged. 90 days hot storage, 1 year archive, 7 years legal hold.

Security Architecture

Defense-in-Depth

  • Layer 1: Network Security (VPC, DDoS protection)
  • Layer 2: Application Security (Input validation)
  • Layer 3: Data Security (AES-256, TLS 1.3)
  • Layer 4: Agent Security (Code signing)

Technology Stack

  • Backend: Python 3.11+, FastAPI, PostgreSQL + Redis
  • Frontend: React 18+, Redux, Material-UI
  • Agent: Go (compiled, <10MB)
  • DevOps: Docker, Kubernetes, Terraform

Scalability

Target: 10,000 concurrent users, 50,000 servers, 1M queries/day, 99.9% uptime

Disaster Recovery

RTO: 15 minutes. RPO: 5 minutes. Multi-region deployment with 3x redundancy.
